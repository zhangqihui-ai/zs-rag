像真实官网一样嵌入对话

本页演示 iframe / 气泡嵌入 /chat/embed。要直接聊天：在下面脚本里的 ZS_RAG_DEMO_CONFIG 填入控制台「嵌入网站」复制的明文密钥（以 zs_rag_embed_ 开头），保存后整页刷新；也可用地址栏 ?api_key=…&space=… 覆盖配置。

安全提示：不要把嵌入 API Key 写进公开仓库或发给不可信第三方；生产环境强烈建议密钥仅保存在服务端，由后端注入 Authorization: Bearer …，以免泄露产生模型调用费用。
